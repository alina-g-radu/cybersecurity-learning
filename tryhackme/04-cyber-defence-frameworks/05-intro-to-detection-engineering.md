# Introduction to Detection Engineering

## What I learned

- There are different types of detection approaches:
  - Environmental-based detection  
    → focuses on configuration and normal behavior (baselines)
  - Threat-based detection  
    → focuses on indicators and attacker behavior

---

## Key concepts

- Detection engineering  
  → the continuous process of creating, testing, and improving detection rules to identify malicious activity or misconfigurations

- Configuration  
  → understanding the current environment and infrastructure to detect misconfigurations

- Modelling  
  → defining normal system behavior (baseline) and detecting deviations

- Indicator-based detection  
  → relies on IOCs such as IPs, hashes, or domains

- Threat-based detection  
  → focuses on attacker behavior using TTPs (Tactics, Techniques, and Procedures)

---

## Why it matters in SOC

- Detection engineering helps SOC teams identify threats more effectively and reduce false positives.
- It enables analysts to move from simple indicator-based detection to more advanced behavior-based detection.
