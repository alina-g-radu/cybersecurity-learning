# Junior Security Analyst

## What I learned
- Junior Security Analyst is the first one to analyze the SOC alerts.
- Tasks performed by the Junior Security Analyst include monitoring and investigating security alerts, participating in SOC events, cooperating with the team and keep informed about new attacks and defenses.
- A SOC includes multiple roles such as SOC Analyst (Level 1/2/3), SOC Engineer, SOC Manager, and Incident Responder.
- Praticed with SIEM Dashboard and analyzed alerts.

## Key concepts
- Junior Security Analyst (SOC Level Analyst) works in the 24/7 SOC team.
- SOC (Security Operations Center) is responsible for detecting and responding to security incidents.

## Why it matters in SOC
- The Junior Security Analyst is responsible for initial alert triage and helps identify whether an alert is a true threat or a false positive.
