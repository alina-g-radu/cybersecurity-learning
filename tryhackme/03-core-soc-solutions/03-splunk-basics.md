# Splunk Basics

## What I learned

- Splunk consists of three main components: Forwarder, Indexer, and Search Head.

## Key concepts

- Splunk is a SIEM platform used for collecting, analyzing, and visualizing machine data.
- Splunk Forwarder: Agent installed on endpoints that collects and sends data to Splunk.
- Splunk Indexer: Processes and stores incoming data for searching and analysis.
- Search Processing Language (SPL): Query language used to search and analyze indexed data.

## Why it matters in SOC

- Splunk is one of the most widely used SIEM tools in SOC environments for monitoring and investigating security events.

## Tools / activities used

- Practiced using Splunk to search and analyze log data.

## Personal notes

- Understanding Splunk components helps clarify how log data flows from endpoints to analysis in a SOC environment.
