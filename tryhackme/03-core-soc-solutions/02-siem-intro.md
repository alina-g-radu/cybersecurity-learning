# Introduction to SIEM

## What I learned

- There are two main types of logs: host-centric logs and network-centric logs.
- SIEM provides centralized log collection, normalization, correlation, real-time alerting, dashboards, and reporting.

## Key concepts

- Security Information and Event Management (SIEM): A system that collects data from multiple sources, standardizes log formats, correlates events, and helps detect potential threats.

## Why it matters in SOC

- SIEM provides centralized visibility into security events, allowing analysts to detect, investigate, and respond to threats more efficiently.

## Tools / activities used

- Analyzed alerts in a SIEM environment.

## Personal notes

- SIEM systems make it easier to identify patterns and suspicious activity by correlating data from multiple sources in one place.
